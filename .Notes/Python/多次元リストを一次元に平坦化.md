## 多次元リストを一次元に平坦化

#### itertools.chain.from_iterable()

```py
import itertools

l_2d = [[0, 1], [2, 3]]

print(list(itertools.chain.from_iterable(l_2d)))
# [0, 1, 2, 3]
```

```py
t_2d = ((0, 1), (2, 3))

print(tuple(itertools.chain.from_iterable(t_2d)))
# (0, 1, 2, 3)
```

#### sum()

```py
l_2d = [[0, 1], [2, 3]]

print(sum(l_2d, []))
# [0, 1, 2, 3]
```

```py
t_2d = ((0, 1), (2, 3))

print(sum(t_2d, ()))
# (0, 1, 2, 3)
```

#### リスト内包表記

```py
matrix = [[1, 2, 3], [4, 5, 6], [7, 8, 9]]

flat = [x for row in matrix for x in row]
print(flat)
# [1, 2, 3, 4, 5, 6, 7, 8, 9]
```
